# 업적 최종 목록

이 문서는 출시 버전에 실제로 도입할 업적의 최종 목록이다.
이 문서의 업적 ID를 기준으로 이후 업적 카탈로그 코드, DB 진행도 관리, 보상 수령 로직을 구현한다.

기본 원칙:
- 숨겨진 업적 없음
- 이벤트 업적 없음
- 출시 후 추가 업적 계획 없음
- 출시 전에 모든 업적을 확정해서 넣는다
- 업적 정의는 코드에서 관리하고, 진행도 및 수령 여부는 DB에서 관리한다
- 보상 수령과 `모든 보상 획득`은 서버에서만 처리한다

## 15. 구현 메모

### 보상 수령 방식
- 모든 업적 보상은 서버에서만 지급한다
- `모든 보상 획득`도 서버에서 처리한다

### 수령 상태 저장
- `claimed` 여부는 DB에서 관리한다
- 클라이언트는 표시만 하고, 실제 수령 가능 여부는 서버가 판단한다

### 능력대전 특수 업적 판정 메모
- 풀피 승리: 승리 시점 HP와 정상 승리 여부 필요
- 스킬 마무리: 마지막 유효 피해 스킬 ID 필요
- 방어 업적: 방어 성공 이벤트 기록 필요
- 유틸 업적: 사용 횟수 누적 필요
