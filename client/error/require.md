# 요청: 로비 화면을 확대/축소(접근성 포함)에도 깨지지 않도록 레이아웃 재설계

현재 로비 화면은 브라우저 확대(예: 150%, 200%) 또는 OS 접근성 설정(글꼴 크기 증가) 시 레이아웃이 무너지고, 일부 UI(특히 상단 로고)가 잘리는 문제가 있습니다.

이 문제를 해결하기 위해, 로비 레이아웃을 **확대/축소에 강한 유동형 구조**로 재설계해주세요.

---

## 🎯 목표

- 100% ~ 200% 이상의 확대/축소에서도 레이아웃이 깨지지 않을 것
- 텍스트, 버튼, 카드 크기가 커져도 **잘리지 않고 자연스럽게 재배치될 것**
- 필요 시 세로 스크롤을 허용 (한 화면 고정 금지)
- 다양한 해상도 (모바일 / 태블릿 / 데스크탑) + 확대 환경 모두 대응
